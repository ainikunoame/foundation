# git
> my rules for using git and help

## rules
### initial
- 大文字
- 開始は以下
  - Add : 追加
  - Fix : 修正
  - Update : 修正
  - Remove : 削除、除去
  - Use : 使用
  - Don't : できなく、しなくする
  - Make : 作成、改良
  - Move : 移動
  - Change : 変更
  - Rename : 名前の変更
  - Improve : 向上
  - Avoid : 回避
  - Allow : 許可
  - Implement : 埋め込む
