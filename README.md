# git
my rules for using git, and help

## rules
### commit massage
- 先頭は大文字
- 開始はできるだけ以下から
  - Add : 追加
  - Fix : 問題を修正
  - Edit : 編集
  - Update : 更新
  - Remove : 削除、除去
  - Use : 使用
  - Don't : できなく、しなくする
  - Make : 作成、改良
  - Move : 移動
  - Change : 変更
  - Rename : 名前の変更
  - Improve : 向上
  - Avoid : 回避
  - Allow : 許可
  - Implement : 埋め込む
