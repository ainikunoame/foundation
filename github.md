### commit massage

| 先頭   | 意味       | 
| ------ | ---------- | 
| Add    | 追加       | 
| Fix    | 問題の修正 | 
| Edit   | 編集       | 
| Update | 更新       | 
| Remove | 削除、除去 | 
| Rename | 名前の変更 | 
| Use    | 使用       | 
| Make   | 作成、改良 | 
| Change | 変更       | 
| Avoid  | 回避       | 
| Allow  | 許可       | 
| Don't  | 否定       |
